# 트럼펫 운지법 배우기 애니메이션 모바일 웹앱

B♭ 트럼펫 기보음 F♯3–C6의 운지와 손가락·피스톤 움직임을 학습하는 오프라인 우선 단일 페이지 앱이다. 녹음 샘플 없이 입술 버즈 배음, 텅잉, 브레스, 벨 포먼트, 비브라토와 짧은 룸 반사를 합성하는 고품질 Web Audio 트럼펫 엔진을 포함한다.

## 실행

최종 배포본 `index.html`을 브라우저에서 직접 열거나 이 폴더를 정적 서버로 제공한다. 앱은 외부 CDN, 웹폰트, 원격 음원을 사용하지 않는다.

## 개발 명령

```text
npm install
npm run build
npm test
npm run test:e2e
npm run validate
npm run qa
```

## 주요 입력

- 화면 숫자 1–8: 현재 분류의 계이름
- Ctrl+1–8: C4–C5 낮은 계이름
- Shift+1–8: C5–C6 높은 계이름
- Alt+1–8: C4 기준 반음(3·7 비활성)
- Space: 현재 음 다시 듣기
- R: 운지 동작 다시 보기

일반 브라우저가 Ctrl+숫자를 탭 전환에 사용할 수 있으므로 화면 입력은 항상 동등하게 제공된다.

## 트럼펫 음색

현재 음 카드의 `콘서트 오픈 · mf` 배지나 설정 버튼을 누르면 다음 조합을 바꿀 수 있다.

- 음색: 콘서트 오픈, 웜 브라스, 컵 뮤트
- 다이내믹: 부드럽게 p, 자연스럽게 mf, 힘차게 f
- 음높이: 기보음 또는 B♭ 실음

엔진 구조, 프리셋과 실제 Chrome 신호 검증값은 [AUDIO-ENGINE.md](./AUDIO-ENGINE.md)에 정리되어 있다.

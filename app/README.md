# 트럼펫 운지법 배우기 애니메이션 모바일 웹앱

B♭ 트럼펫 기보음 F♯3–C6의 운지와 손가락·피스톤 움직임을 학습하는 오프라인 우선 단일 페이지 앱이다.

## 실행

최종 배포본 `index.html`을 브라우저에서 직접 열거나 이 폴더를 정적 서버로 제공한다. 앱은 외부 CDN, 웹폰트, 원격 음원을 사용하지 않는다.

## 개발 명령

```text
npm install
npm run build
npm test
npm run test:e2e
npm run validate
npm run qa
```

## 주요 입력

- 화면 숫자 1–8: 현재 분류의 계이름
- Ctrl+1–8: C4–C5 낮은 계이름
- Shift+1–8: C5–C6 높은 계이름
- Alt+1–8: C4 기준 반음(3·7 비활성)
- Space: 현재 음 다시 듣기
- R: 운지 동작 다시 보기

일반 브라우저가 Ctrl+숫자를 탭 전환에 사용할 수 있으므로 화면 입력은 항상 동등하게 제공된다.
